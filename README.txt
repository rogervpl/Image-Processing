O código foi testado no Windows 10.


O programa roda em Python 3. 




As dependencias são: OpenCV, numpy, matplotlib PyQt5 e liac-arff.
Para instalar essas bibliotecas, é necessário utilizar os seguintes comandos
num terminal com acesso elevado:



pip3 install opencv-python

pip3 install numpy

pip3 install matplotlib

pip3 install pyqt5
pip install liac-arff



O programa é dividido em três arquivos:


main.py - Contém o código que chama a interface e a biblioteca com as funções utilizadas.

biblioteca.py - Contém as principais funções utilizadas no programa.

output.py - Contém o código gerado pelo qtDesigner para produzir a interface.


Obs.: Algumas funcionalidades necessitam de input no terminal.
